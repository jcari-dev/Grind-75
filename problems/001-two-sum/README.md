# Two Sum

🔗 [LeetCode URL](https://leetcode.com/problems/two-sum)

## Approach
- Use a hash map (dictionary in this case) to store previously seen numbers.
- For each number, check if the complement exists in the map.

## Complexity
- ⏱️ Time: O(n)
- 🧠 Space: O(n)

## Data
- Category: Array
- Difficulty: Easy
